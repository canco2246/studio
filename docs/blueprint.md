# **App Name**: TagSync Pro

## Core Features:

- Item Scan Interface: A web-based interface to input or simulate RFID tag scans to quickly add new assets, linking physical tags to digital records received via API or manual entry.
- Asset Details Editor: Form-based interface to view, edit, and update comprehensive details for individual assets, including custom attributes, statuses, and locations.
- Asset Catalog View: A browsable and sortable list displaying all managed assets with key information, pagination, and quick filters for an overview.
- Advanced Inventory Search & Filter: Powerful search capabilities to find assets by various criteria (name, ID, category, location) with multi-field filtering options.
- Unique Asset Identifier Enforcement: Backend logic and UI validation to prevent the creation of duplicate asset records based on unique identifiers like RFID tag IDs, using a database (e.g., PostgreSQL).
- AI-Powered Description Tool: A generative AI tool to assist users in creating detailed and consistent descriptions for new or existing assets based on minimal input like keywords or categories.

## Style Guidelines:

- Primary color: A vibrant, professional Electric Blue (#1A67F9) to convey innovation and efficiency.
- Background color: A subtle and light off-white with a hint of blue (#EEF2F9) for a clean, modern base.
- Accent color: A striking Vibrant Cyan (#05CAEB) for calls to action, highlights, and interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and impactful appearance.
- Body font: 'Inter' (sans-serif) for clear readability and a professional, objective feel in longer text blocks.
- Use sleek, minimalist line icons that clearly represent asset management concepts such as tags, inventory, search, and data, aligning with the modern aesthetic.
- Clean, grid-based layouts with ample white space to emphasize content and facilitate intuitive navigation within a professional, light theme.
- Subtle, fluid transitions and micro-interactions for state changes, data loading, and modal displays, enhancing the user experience without being distracting.
